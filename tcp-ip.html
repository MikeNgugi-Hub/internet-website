Definition

TCP/IP stands for Transmission Control Protocol/Internet Protocol. It is a whole set of related protocols and tools that help computers to communicate with each other. The Internet operates using TCP/IP as its foundational communication standard — without it, computers across different networks would not be able to exchange data reliably.

Protocols That Use TCP/IP

Several protocols are built on top of TCP/IP and are used on the Internet every day. Some important examples include:

  • SMTP (Simple Mail Transfer Protocol) — used for sending email messages between computers.
  • FTP (File Transfer Protocol) — allows files to be easily copied to and from remote sites.
  • HTTP (Hypertext Transfer Protocol) — used by the World Wide Web to send and receive web pages.

IP Addresses

Every machine connected to the Internet is assigned a unique numerical address called an IP address. This address is what allows data to be sent to the correct destination across the network.

IPv4 is a 32-bit binary number normally represented as four decimal values called octets. Each octet represents 8 bits in a range from 0 to 255, separated by decimal points. This method of notation is called dotted decimal notation. For example: 216.27.61.137

IPv6 addresses are written as eight 4-digit hexadecimal numbers separated by colons, each representing 16 bits, giving a total of 128 bits. IPv6 was introduced because the world was running out of available IPv4 addresses due to the rapid growth of Internet-connected devices.

Why Are IP Addresses Important?

To guarantee world-wide unique addresses, IP addresses are licensed from the Network Information Center (NIC). An IP address together with its subnet mask performs the following functions:

  • Enables the system to process the sending and receiving of data packets.
  • Specifies the device's local address on the network.
  • Specifies a range of addresses that share the same cable or network as the device.
