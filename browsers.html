Definition

Computers that read Web pages are called Web clients. Web clients view pages using a program called a Web browser. A web browser is a piece of software that runs on your computer and allows you to view Web pages.

More specifically, a web browser is a software application used to locate, retrieve and display content on the World Wide Web, including Web pages, images, video and other files. As a client/server model, the browser is the client that runs on a computer, contacts the Web server and requests information.

How does a browser fetch Web Pages?

A browser fetches a Web page from a server by sending a request. A request is a standard HTTP request that contains a page address. A page address looks like this:

http://www.strathmore.edu/ams

The browser sends this request to the appropriate web server, which then locates the requested resource and sends it back to the browser to display.

How does a browser display web pages?

All Web pages contain instructions that tell the browser how to display their content. The browser reads these instructions and renders the page accordingly. The most common display instructions are called HTML tags.

For example, an HTML tag looks like this:

<p>This is a Paragraph</p>

The browser reads these tags and uses them to format and present the content correctly on the screen for the user to see.
