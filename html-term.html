What is HTML?

HTML stands for Hypertext Markup Language. It is the standard language used to represent textual and image content on the World Wide Web. Every web page you see in a browser is built using HTML. It provides the basic structure and content of a web page through the use of tags and elements.

Key Characteristics of HTML

Represents textual and image content: HTML is used to structure and display both text and images on a web page. It tells the browser what content to show and how to organize it.

Platform independent: HTML is platform independent, meaning that the text and content are encoded in a way that they can be displayed on a wide range of computers and devices. It does not matter whether the user is on Windows, Mac, Linux or a mobile device — the browser will read and display the HTML correctly.

How HTML Works

HTML uses a system of tags to mark up content. Tags tell the browser how to display each piece of content on the page. Tags are written inside angle brackets and usually come in pairs — an opening tag and a closing tag. For example:

  • <h1>This is a heading</h1> — displays a large heading
  • <p>This is a paragraph</p> — displays a paragraph of text
  • <img src="image.jpg"> — displays an image
  • <a href="page.html">Click here</a> — creates a hyperlink

HTML and the World Wide Web

HTML is one of the three core technologies of the World Wide Web alongside HTTP and URLs. While HTTP handles the transfer of data between the client and server, and URLs identify the location of resources, HTML is responsible for defining the actual content and structure of every web page. Without HTML there would be no web pages as we know them today. It is the foundation on which all web development is built and will be covered in greater detail later in this course.
