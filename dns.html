What is DNS?

A domain name is a unique name that identifies a website. It is an alpha-numeric representation of an IP address where the characters are separated by dots. For example: www.strathmore.edu

IP addresses are not user friendly and could easily cause typing errors. The Domain Name System (DNS) was therefore created so that people would not have to remember several confusing numbers. Domain names enable short, alphabetical names to be assigned to IP addresses, making them easier to remember and more informative and convenient to people.

Structure of a Domain Name

Domain names are composed of several different parts and are read from right to left. A domain name contains the following information:

  • The name of the organization
  • The type of organization
  • The country
  • The particular computer or network

For example, the domain name www.cnn.com has three parts:

  • www — the host name, also referred to as the machine name
  • cnn — the domain name, also called the mid-level domain
  • com — the top-level domain (TLD)

Top Level Domains (TLD)

At the far right of the domain name you will find what is called a Top Level Domain (TLD), also known as the domain name extension. These are the extensions such as .com, .org, .net and others that appear after the final period in the domain name. The TLD is assigned according to what kind of domain it represents.

Some common examples of TLDs include:

  • .com — commercial organizations
  • .org — non-profit organizations
  • .edu — educational institutions
  • .gov — government bodies
  • .net — network providers
  • .ke — websites based in Kenya

Second Level Domains

The second level domain is the part you are most familiar with. It appears after the www but before the top-level domain. This is the part of the domain name where the greatest level of customization is possible — it is typically the name of the organization or brand. For example, in www.mail.yahoo.com, the second level domain is yahoo.

Why DNS Matters

Without DNS, every time you wanted to visit a website you would have to type a long numerical IP address such as 216.27.61.137 instead of a simple name like www.google.com. DNS acts as the Internet's phone book — it silently translates the domain name you type into the correct IP address so your browser can find and load the right website.
